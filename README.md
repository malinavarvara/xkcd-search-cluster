# XKCD Search Cluster

Комплексная система для индексации и поиска по комиксам XKCD, построенная на базе микросервисной архитектуры. Проект включает в себя полноценный ETL-процесс (парсинг, обработка, сохранение) и современный веб-интерфейс для взаимодействия с данными.

## Архитектура системы
Проект разделен на несколько независимых компонентов, работающих в едином Docker-кластере:

Web-сервис: Основной интерфейс взаимодействия. Генерирует страницы поиска и предоставляет административную панель.

API Gateway: Центральный узел, координирующий запросы между фронтендом и внутренними сервисами. Реализует REST API и авторизацию.

Search Service: Движок, отвечающий за быстрый поиск по текстовым индексам.

Update Service: Автономный воркер, который сканирует xkcd.com, скачивает новые комиксы и обновляет базу данных.

Words Service: Модуль лингвистического анализа. Отвечает за нормализацию слов (стемминг) и очистку от «шума».

PostgreSQL: Реляционное хранилище для структурированных данных и полнотекстового поиска.

## Основные возможности
Умный поиск: Поиск по фразам с учетом морфологии слов (благодаря нормализации в words service).

Admin Dashboard: Защищенная панель управления (авторизация по токену).

Live Monitoring: Отслеживание статуса индексации (IDLE/RUNNING) и актуального количества комиксов в БД в реальном времени.

Database Management: Возможность запустить обновление базы или её полную очистку (Drop DB) через веб-интерфейс.

## Качество кода и тестирование
Линтинг: Код прошел проверку golangci-lint (конфигурация включает errcheck, gosimple и др.).

Тесты: В проект включены модульные тесты и интеграционный тест-контейнер для проверки взаимодействия всех узлов кластера.

CI/CD Friendly: Все настройки вынесены в переменные окружения и config.yaml.
